# 有限会社ネオ・ドリーム ウェブサイト

大阪のアクアリウムレンタル・空間演出会社 NeoDream のウェブサイトです。  
GitHub Pages でそのまま公開できます。

---

## フォルダ構成

```
neodream/
├── index.html        ← メインページ（すべてのコンテンツ）
├── images/           ← 画像フォルダ（下記リストを参照）
│   ├── logo.png
│   ├── hero1.jpg
│   ├── hero2.jpg
│   ├── hero3.jpg
│   └── ... （以下参照）
└── README.md
```

## お問い合わせフォームの設定

このサイトは **Formspree**（無料）でフォームを動作させられます。

1. [https://formspree.io/](https://formspree.io/) で無料アカウントを作成
2. 「New Form」でフォームを作成 → フォームIDを取得（例：`xyzabcde`）
3. `index.html` の以下の箇所を書き換えてください：

```html
<!-- 変更前 -->
<form class="form-grid" action="https://formspree.io/f/YOUR_FORM_ID" method="POST">

<!-- 変更後（例） -->
<form class="form-grid" action="https://formspree.io/f/xyzabcde" method="POST">
```

## GitHub Pages への公開手順

```bash
# 1. このフォルダで Git を初期化
cd neodream
git init

# 2. 全ファイルを追加
git add .
git commit -m "Initial commit: NeoDream website"

# 3. GitHub にリポジトリを作成後、リモートを追加
git branch -M main
git remote add origin https://github.com/あなたのユーザー名/neodream.git
git push -u origin main

# 4. GitHub の Settings → Pages → Branch: main → Save
# → https://あなたのユーザー名.github.io/neodream/ で公開されます
```

### 独自ドメイン（neodream.jp）を使う場合

`images/` フォルダと同じ階層に `CNAME` ファイルを作成し、ドメイン名だけ記述します：

```
neodream.jp
```

その後、ドメインの DNS に以下を設定します：
- `A レコード`: `185.199.108.153` / `185.199.109.153` / `185.199.110.153` / `185.199.111.153`
- または `CNAME レコード`: `あなたのユーザー名.github.io`

## 更新方法

コンテンツを変更したい場合は `index.html` をテキストエディタで開いて編集してください。  
画像を差し替えたい場合は `images/` フォルダ内のファイルを同じ名前で上書きしてください。

変更後は以下のコマンドで GitHub に反映されます：

```bash
git add .
git commit -m "Update: 変更内容のメモ"
git push
```
